## Installation
1. Скачайте .asl файл
2. Откройте LiveSplit
3. Edit your layout...
4. Добавьте -> "Control –> Scriptable Auto Splitter"
5. Выберите путь к файлу ASL, нажав кнопку "Browse"..."
6. Выберите необходимый AutoSplitter
7. Сохраните изменения и наслаждайтесь!
